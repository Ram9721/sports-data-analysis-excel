# sports-data-analysis

## Project Overview
This is a personal data analysis project focused on analyzing sports-related data using Advanced Microsoft Excel. The project demonstrates practical data analyst skills such as data cleaning, logical analysis, and the use of advanced Excel formulas to derive insights from structured datasets.

---

## Tools & Technologies
- Microsoft Excel (Advanced)  
- Advanced Excel formulas:
  - IF, IFS  
  - SUMIFS, COUNTIFS  
  - VLOOKUP / XLOOKUP  
  - INDEX & MATCH  
  - TEXT, DATE functions  
- Data sorting, filtering, and validation
- Pivot tables  

---

## Project File
- **Sports Data Analysis Project.xlsx**  
  Contains raw data, cleaned data, calculated fields, and analysis performed using advanced Excel techniques.

---

## Key Tasks Performed
- Cleaned and standardized sports data to ensure consistency  
- Used advanced Excel formulas to calculate performance metrics  
- Applied logical functions to classify and segment data  
- Performed conditional analysis using multi-criteria formulas  
- Organized datasets for accurate and efficient analysis
- Pivot tables for deeper analysis 

---

## Key Insights
- Identified performance differences across various categories  
- Extracted meaningful insights using calculated fields  
- Improved data accuracy through structured cleaning and validation
